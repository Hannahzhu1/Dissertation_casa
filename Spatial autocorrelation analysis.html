1.Hypothesis

Null hypothesis: The noise complaint has not spatial clustered pattern.

Alternative hypothesis: The noise complaint has spatial clustered pattern.

Set our significance level : 0.05

6.Analysis

Moran I analysis

Morans I requires a spatial weight list type object.

Morans I falls between -1 and 1. And the results could be interpreted as follow: 1 = clustered, 0 = no pattern, -1 = dispersed.

#create a spatial weight list
NY_LIST <- NY_nb %>%
  nb2listw(., style="W")

#calculate for the densities
I_NY_Raw_Density <- points_sf_joined %>%
  pull(density) %>%
  as.vector()%>%
  moran.test(., NY_LIST)

I_NY_Raw_Density

From the results, The Morans I statistic = 0.51, which shows that we have some distinctive clustering.

Gearys C Analysis

Gearys C Analysis is used to see whether similar values or dissimilar values are clustering.

Gearys C falls between 0 and 2, and the results could be interpreted as follow: 1 means no spatial autocorrelation, <1 - positive spatial autocorrelation or similar values clustering, >1 - negative spatial autocorreation or dissimilar values clustering.

C_NY_Raw_Density <- points_sf_joined %>%
  pull(density) %>%
  as.vector()%>%
  geary.test(., NY_LIST)

Getis Ord analysis

General G is used to see whether high or low values are clustering.

If G > Expected = High values clustering; if G < expected = low values clustering
